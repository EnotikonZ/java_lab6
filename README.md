## Что реализовано

### Задание 1. Аннотации (1–6)
1. **@Invoke** — отмечает методы для автоматического вызова обработчиком.
2. **@Default** — хранит обязательный параметр `value: Class` (класс по умолчанию) и выводит его.
3. **@ToString** — формирование строкового представления объекта с учётом `Mode.YES/NO`.
4. **@Validate** — хранит массив типов `Class[]` и обработчик выводит перечисление.
5. **@Two** — два обязательных свойства: `first: String`, `second: int`.
6. **@Cache** — необязательный массив областей кеширования `String[]`.

### Задание 2. Тестирование (вариант 9)
- **№4:** тест корректности извлечения типов из `@Validate`, а также проверка исключения при пустом массиве.
- **№6:** тест валидации `@Two` при некорректных свойствах (`first=""`, `second<0`) с использованием `assertThrows()`.
